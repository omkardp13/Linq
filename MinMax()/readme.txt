Min():
The Min() method returns the smallest value from a collection or based on a property if you're working with complex types.

Syntax:

var minValue = collection.Min();
var minValue = collection.Min(x => x.SomeProperty);

-------------------------------------------------------------------------------------------------------------------------------------------------

Max():
The Max() method returns the largest value from a collection or based on a property if you're working with complex types.

Syntax:
var maxValue = collection.Max();
var maxValue = collection.Max(x => x.SomeProperty);
